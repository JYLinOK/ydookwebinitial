1. IntelliJ IDEA


Kotlin 是 JetBrains 开源的，而 JetBrains 的代表作就是 IntelliJ IDEA 。

可以说，IntelliJ IDEA 应为 Kotlin 第一官方 IDE。

2. Android Studio


Android Studio 是由谷歌公司推出的基于 Andriod 系统应用开发的集成开发环境 (IDE)。

Kotlin 目前主要的应用情景是用于进行 Andriod 应用开发，而 Android Studio 是 Andriod 应用开发默认 IDE。所以 Kotlin 与 Andriod Studio 的结合使用是目前官方推荐的用于开发 Andriod 应用的最佳组合。

自2017年5月起，Android Studio 已经全面支持 Kotlin，包括支持标准组件和 Android 模板。

3. Eclipse


Eclipse 是老牌的IDE，功能强大，可以进行基于 Andriod 系统应用的开发。

Eclipse 是免费的开源软件，全球用户数量巨大。

Eclipse 只是一个软件框架与服务，其强大的功能通过安装其对应的丰富的插件实现。

4. TryKotlin


TryKotlin 是 JetBrains 为 Kotlin 开发了一个官方的在线的网页版 Kotlin IDE。

TryKotlin 功能强大，界面简单，可以实现随时随地的 Kotlin 编程。

7. Sublime Text 3


Sublime Text 3 是一款具有精美用户界面及丰富拓展功能的代码的跨平台代码编辑器。

Sublime Text 3 支撑多编程语言。

Sublime Text 3 的官方定位是：一款用于代码、标记和散文的精致文本编辑器。

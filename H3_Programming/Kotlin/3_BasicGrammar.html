1. Hello World

Hello World 是业内普遍认可的编程语言的标准入门程序。

通过 Hello World 可以快速对某种编程语言的编程语法风格及大致的特征有所认识。

下面是 Kotlin 的 Hello World 程序代码：

    println ("Hello World")
}





2. 包 package

Kotlin 具有 包 的结构。

Kotlin 的源文件应该以包声明开头。

定义包的时候需要在源代码的顶部声明。包的声明通过 package 关键字实现。

使用引入其他包时，需要引入。包的引入通过 import 关键字实现。


1
2
3
4
5
6 
package my.package

import java.util.*


//...


点击此处转至 包 的详细说明。




3. 函数 fun

函数是编程语言的关键组成部分，是利用编程语言实现功能及方法的基本元素。

Kotlin 的函数通过关键字 fun 定义。


Kotlin 函数声明

Kotlin 函数声明主要形式为：

1
2
3 
fun fun_name (x:Int) : Int{
    return x+x
}


从上述代码中可以看到:

1. Kotlin 语句不需要加 ; 号结尾；
2. Kotlin 中的函数使用关键字 fun 进行定义；
3. 上述 fun_name 代表被定义的函数名；
4. fun_name 后面的，括号里面的 x 代表传输函数 fun_name 中的参数，x 的数据类型使用 : 号声明为 Int 数据类型；
5. Kotlin 中的 Int 数据类型关键字的首字母 I 为大写；
6. 函数圆括号右边的 : 号后面的 Int 为函数 fun_name 的函数返回值的数据类型；
7. Kotlin 的函数体通过大括号 { } 包括起来。



Kotlin 函数声明定义的其他形式

1. 带两个或多个传入参数的函数的定义

1
2
3
4
5
6
7
8
9 
fun sum(x:Int, y:Int) : Int{
    return x+y
}


fun  main() {
    print ("sum of 3 and 5 is ")
    println (sum(3, 5))
}


上述代码中反映了以下知识:

1. Kotlin 程序代码具有著名的程序入口 main() 函数；
2. 在定义函数的时候，如果要传入两个以上的参数，圆括号内的多个参数需要使用 , 号隔开；
3. 每个传入参数都应该被定义具体的数据类型；
4. 在主函数 main() 函数中调用被声明定义的函数，实现被定义函数的功能；
5. Kotlin 中 print() 函数用于直接向控制台输出数据，输出后不换行；
6. Kotlin 中 println() 函数用于直接向控制台输出数据，输出后换行。



2. 定义将表达式作为函数体，省略函数的返回值，返回值类型自动推断的函数

1 
fun sum(x:Int, y:Int) = x + y


上述代码中反映了以下要点:

1. Kotlin 的函数可以被定义成一个简单的表达式，这样的表达式一般是算数表达式；
2. 可以将上述通过表达式定义函数的 Kotlin 称为 表达式函数 ；
3. 表达式函数的全部函数内容就是一个表达式，一般可以用一行代码表示；
4. 表达式函数必定是具有返回值的函数，函数的返回值就是表达式的运算结果；
5. 表达式函数的算术组成部分应该包括函数定义的所有传入参数。



3. 神奇的 ${}

1
2
3
4
5
6 
fun main(){
    val a = 1
    val b = 2
    
    print ("The sum of $a and $b is ${a + b}")
}


执行上述代码中，输出结果为:
3

上述代码中反映了以下要点:

1. Kotlin 的函数中，$ 符号的功能相当于算数表达运算符；
2. $ 后面可以跟一个变量，此时表示取该变量的值；
3. $ 后面可以跟一个算术表达式，此时表示取该算数表达式的运算结果；
4. $ 后面跟一个算术表达式，该算数表达式需要使用 { } 包括起来 。



4. 使用 Unit ，定义返回无类型数据的函数

1
2
3
4
5
6
7 
fun Funit(a:Int, b:Int) : Unit{
    print ("The sum of $a and $b is ${a + b}")
}

fun main(){
    Funit(1,2)
}


执行上述代码中，输出结果为:
3

上述代码中反映了以下要点:

1. Kotlin 的函数数据类型定义中，使用 Unit 可以定义函数返回的数据为 无类型 的数据；

2. 只要无类型返回值的函数具有逻辑可行的运算结果，Unit 定义的返回值就能够被正确地返回。



5. 也可以省略 Unit ，定义直接输出返回无类型返回值的函数

fun Funit(a:Int, b:Int) {
    print ("The sum of $a and $b is ${a + b}")
}

fun main(){
    Funit(1,2)
}


执行上述代码中，输出结果仍然为:
3

上述代码中反映了以下要点:

1. Kotlin 的函数数据类型定义中，可以不指定返回值的具体数据类型，此时的函数返回值自动被定义为 Unit 类型。

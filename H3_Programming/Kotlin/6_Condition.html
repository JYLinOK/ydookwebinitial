条件分支  条件判断语句

Kotlin 具有下列条件判断语句：
1. if 条件判断；
1. when 条件判断。



if 条件判断

1. 简单的 if 语句

1
2
3 
if(expression == true){
    implement_sentences...
}



2. if - else 语句

1
2
3
4
5
6
7 
if(expression1 == true){
    implement_sentence_1 
    ...
} else(expression2 == true){
    implement_sentence_2 
    ...
}



3. 多重 if - else 语句
Kotlin 不支持 elif 组合语句，所以使用 else 与 if 组合使用。

1
2
3
4
5
6
7 
if(expression1 == true){
    implement_sentence_1 
    ...
} else if(expression2 == true){
    implement_sentence_2 
    ...
}



3. 使用 if - else 语句代替三元运算符
Kotlin 为了简化编程语法，取消了 Java 语言中的三元预算符：

变量 = 条件判断语句 ？ 取值A : 取值 B

Kotlin 使用 if - else 语句代替三元运算符的格式为：

if (条件判断语句) (表达式1) else (表达式2)




when 条件判断

1. 简单的 when 语句
简单的 when 语句与简单 if 语句实现的功能一致：

1
2
3
4 
when(expression == true){
    implement_sentences
    ...
}



2. when - else 语句
1. Kotlin 中没有 switch - default 判断选择语句。而是使用更加简化的 when - else 判断选择语句代替。
2. 在 Kotlin when - else 判断选择语句中，使用 -> 号代替switch - default 判断选择语句中的 : 号。

Kotlin 的 when - else 判断选择语句基本语法结构如下：

1
2
3
4
5
6
7
8 
when(expression result){
    condition1 -> implement1
    condition2 -> implement2
    condition3 -> implement3
    ...
    
    else -> reserve implement
}




3. Kotlin when - else 语句与 Java switch - default 语句比较
1. Kotlin 的 when - else 语句 省略了 case 关键字，直接使用条件表达式。

2. Java 在使用 switch - default 语句时有个限定 : case 后面只能跟常量，不能跟变量。而 Kotlin whne - else 条件表达式中可以是常量，也可以是变量。



4. Kotlin when 语句与 is 关键字结合判断表达式的数据类型
具体使用实例如下：

1

3

when(expression){
    is Int -> implement1
    is Long -> implement2
    is Char -> implement2
    is String -> implement2
    ...
    ...
    else -> reserve implement
}

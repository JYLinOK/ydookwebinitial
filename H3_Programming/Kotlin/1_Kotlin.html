Android 官方支持开发语言


1. Kotlin 编程语言概述

1. Kotlin 是一种用于现代多平台应用编程的 静态编程语言 ，由 JetBrains 开发的。
2. Kotlin 可以编译成 Java 字节码，也可以编译成JavaScript代码，可以方便地在没有安装JVM环境的机器上运行。
3. Kotlin 已经成为 Android 官方支持开发语言，主要用于进行基于 Android 系统的 App开发设计。
4. Kotlin 已经基于 Apache 许可，由 JetBrains 进行开源。

点击访问 Kotlin 官方站 ：
https://kotlinlang.org

点击访问 Kotlin 中文站 ：
https://www.kotlincn.net






2. 使用 Kotlin 编程的优点及好处

1. 简洁: 可以大大减少样本代码的数量；
2. 安全: 可以避免类似于空指针异常等整个类的错误；
3. 互操作性: 可以充分利用 JVM、Android 和浏览器的现有库；
4. 工具友好: 可用任何 Java IDE 或者使用命令行构建。

循环控制  循环控制语句

Kotlin 具有下列条件判断语句：
1. for 循环；
2. while 循环。



for 循环

1. 简单的 for 循环
Kotlin 简单的 for 循环的用法与作用于其他编程语言类似：

1
2
3
4 
for(initialize; condition; increase or decrease){
    implement_sentences
    ...
}



2. for - in 循环遍历
for 循环与关键字 in 结合可以实现遍历功能：

for 循环可以对任何提供迭代器（iterator）的对象进行遍历。

1
2
3
4 
for(item in collection){
    println(item)
    ...
}



Kotlin for-in 语句结合 区间表达式 进行数字区间的遍历：

1
2
3
4 
for(item in 1...3){
    println(item)
    ...
}



Kotlin for-in 语句结合 区间表达式 进行数字区间的遍历：

1
2
3
4 
for(item in 1...3){
    println(item)
    ...
}



Kotlin for-in 语句通过索引遍历数组或者队列，有两种方法：

1. 通过调用数组或者队列的 .indices 属性进行遍历。数组或者队列的 .indices 属性储存的是数组或者队列的下标数组集合。

1
2
3
4 
for(i in array.indices){
    println(array[i])
    ...
}


2. 通过调用数组或者队列的 .withIndex() 函数进行遍历。

数组或者队列的 .withIndex() 函数储存的是数组或者队列的 (下标索引 + 对应内容)。

1
2
3
4 
for((index, value) in array.withIndex()){
    println("the array element at $index position is $value")
    ...
}

注意：for 循环的条件说明的最外层必须用一个大括号括起来。




while 循环

Kotlin 的 while 循环与 do-while 循环与 Java 的 while 循环与 do-while 的类似。


Kotlin 的 while 循环
下面代码展示 Kotlin while 循环的标准使用：

3
4 
while(expression == true){
    implement_sentences
    ...

结构化跳转表达式

Kotlin 具有以下 3 结构化跳转表达式：
1. return : 默认的，直接返回函数运算结果到所处的外层函数或者匿名函数。
2. break : 直接终止整个循环。所处循环的后续部分不再执行。
3. continue : 直接跳出本次循环。所处循环的后续部分将继续执行。
